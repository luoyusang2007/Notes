# 反射图
反射图又叫反弹图/点阵图/网格图（Lattice Diagram / Bounce Diagram）
```

    Source  --->    ============Line============    --->    Load
    V(SUP)-Z(S)-V(S)            Z(0)                   V(L)-Z(L)
                                T(D)=t
            <-ρ(S)-                               -ρ(L)->   
==========================================================================
Time    V(S)    |                                       |   V(L)
0          -    |  \                                    |   0    
          /     |   \------>    v(a)   >------------\   |    /   
t       V(A)    |                                    >  |   <    
          \     |   /------<    v(b)   <------------/   |    \   
2t         >    |  <                                    |   V(A')
          /     |   \------>    v(c)   >------------\   |    /   
3t      V(B)    |                                    >  |   <    
          \     |   /------<    v(d)   <------------/   |    \   
4t         >    |  <                                    |   V(B')
          /     |   \------>    v(e)   >------------\   |    /   
5t      V(C)    |                                    >  |   <    
          \     |   /------<    v(f)   <------------/   |    \   
6t         >    |  <                                    |   V(C')
          /     |   \------>    v(g)   >------------\   |    /   
7t      V(D)    |                                    >  |   <    
          \     |   /------<    v(h)   <------------/   |    \   
...     ...     |               ...                     |   ...

V(S) 是信号源内阻在传输线侧的电压，而非电源侧。所以 t=0 时刻要算V(S) 初始值。
V(S-init)=V(SUP)(Z(0)/(Z(0)+Z(S)))

v(a)=V(S)
v(b)=ρ(L)v(a)
v(c)=ρ(S)v(b)
...

V(A )=v(a)
V(A')=v(a) + v(b)
V(B )=v(a) + v(b) + v(c)
V(B')=v(a) + v(b) + v(c) + v(d)
...
```
